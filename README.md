# Fintrack Dashboard UI:

 Live Demo-https://fintrack-prasad.netlify.app/
 
-About Project:
This is a simple and interactive **Finance Dashboard UI** built using React.js.  
It helps users to track income, expenses, and understand their financial data through charts and insights.

-Features:
- Dashboard with charts  
- Transactions with search & filter  
- Insights for financial analysis  
- Dark / Light mode  
- Add transaction (UI simulation)  
- Responsive design  

-Tech Stack:
- React.js  
- React Router  
- CSS  
- Recharts  

-Run Project:
npm install
npm run dev
